-웹 사용법
1. 검색 창에 경기도 지역 검색 후 오른쪽 하단 추천 의상 정보 보기
2. 홈 화면 모서리 지도 버튼 클릭해서 검색 지역 지도 확인
3. 4개의 홈, 그래프 버튼을 눌러서 날씨 관련 정보 확인
4. 오른쪽 상단에 지메일 버튼을 눌러서, 날씨정보를 전송
5. 나무 버튼을 눌러서 지역 명소 정보 보기
6. 텔레그렘 버튼을 통해 '오늘 뭐입지' 챗봇으로 날씨 정보 물어보기
